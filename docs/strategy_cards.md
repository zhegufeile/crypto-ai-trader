# Strategy Cards

策略卡用于把 KOL 或复盘内容沉淀成结构化知识。

```json
{
  "name": "breakout_follow",
  "market": "bullish",
  "entry_conditions": ["volume_expansion", "oi_rising", "btc_stable"],
  "exit_conditions": ["price_fails_breakout", "funding_too_hot"],
  "risk_notes": ["avoid low liquidity", "avoid news-driven spikes"]
}
```
